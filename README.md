# PLANET FACTS

## Описание

Проект -- сайт с информацией о планетах в нашей солнечной системе. 
Вся информация берется с Википедии и обновляется вслед за исправлениями в Википедии.


### Используемые технологии

- HTML
- CSS
- JS
- БЭМ
- API
